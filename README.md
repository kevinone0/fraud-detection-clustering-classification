# Transaction-Behavioral-Analysis-and-Anomaly-Detection-via-K-Means-Clustering-and-Random-Forest
Proyek Machine Learning untuk melakukan segmentasi pola transaksi nasabah menggunakan K-Means dan membangun model klasifikasi deteksi fraud dengan algoritma Random Forest yang mencapai akurasi 99%.
- [Clustering]_Submission_Akhir_BMLP_Rifqi_Zaghlul.ipynb: Eksplorasi data, pembersihan, dan segmentasi K-Means.  
- [Klasifikasi]_Submission_Akhir_BMLP_Rifqi_Zaghlul.ipynb: Pembangunan model klasifikasi dan optimasi hyperparameter.  
- tuning_classification.h5: Model akhir yang telah melalui proses tuning.
